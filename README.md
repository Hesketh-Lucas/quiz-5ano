# 🎓 Revisão 5º Ano

Plataforma interativa de quizzes para revisão de conteúdos do 5º Ano do Ensino Fundamental.

## 📚 Matérias disponíveis

- História
- Geografia
- Língua Portuguesa *(em breve)*
- Matemática *(em breve)*
- Ciências *(em breve)*

## 🗂️ Estrutura

Os quizzes são organizados por avaliação bimestral (AC1, AC2, AC3 e AC4),
permitindo que os alunos revisem os conteúdos de cada período de forma independente.

## 🚀 Como usar

Acesse o link, escolha a avaliação, selecione a matéria e responda às perguntas.
Cada quiz tem cronômetro, feedback imediato e placar final.

## 💡 Como surgiu

Este projeto nasceu de um pai que queria ajudar a filha a estudar de uma
maneira mais leve e criativa — e que esse esforço pudesse também alcançar
as amigas dela. O que começou como uma ferramenta para uma criança
acabou se tornando uma plataforma aberta para qualquer aluno do 5º Ano.

## 📖 Referências

Os conteúdos foram elaborados com base nos livros didáticos adotados
para o 5º Ano do Ensino Fundamental.

## 🛠️ Tecnologia

HTML, CSS e JavaScript puros — sem dependências externas.
Funciona em qualquer navegador.
